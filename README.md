# 📰 Fake News Detector

## Overview

The **Fake News Detector** is a machine learning web application designed to classify news articles as either **Fake News** or **Real News**. The project aims to demonstrate the practical application of Artificial Intelligence (AI) and Natural Language Processing (NLP) techniques in solving a real-world problem: the spread of misinformation online.

Users can paste the text of a news article into the application and receive a prediction generated by a trained machine learning model. The application provides a simple and user-friendly interface built with Streamlit.

## How the System Works

1. The user enters or pastes a news article into the application.
2. The text is preprocessed using the same techniques applied during model training.
3. The trained TF-IDF vectorizer converts the text into numerical features.
4. The machine learning model analyzes those features.
5. The application displays whether the article is predicted to be **Fake News** or **Real News**.

## Installation

1. Clone or download the project.
2. Install the required dependencies:

```
pip install streamlit pandas scikit-learn joblib
```

3. Run the application:

```
python -m streamlit run app/app.py
```

---

## Expected Model Files

The application expects the following files inside the `models` directory:

* `model.pkl` – Trained machine learning model.
* `vectorizer.pkl` – Saved TF-IDF vectorizer used during training.

Without these files, the interface will run but predictions cannot be generated.
